tutorial #1

 

multiplication
 
 

In this chapter you will learn an easy ways to multiplicate objects.

Preparatory Steps

The desired result requires a few simple preparation steps, which will be explained later.

Insert a matrix hub module (modules > matrix > matrix hub) between the identity matrix and the rotate matrix modules.

Now, select the matrix out connector on top of the hub module and insert a translation matrix.

The preparatory steps are finished and you will now ...

Copy a Matrix Hierarchy

Click on the translate module while holding down the ALT key.

You will see a change in the appearance of the modules in the matrix hierarchy above the translate module: the object modules got a black outline and the value connectors are dressed in a wireframe mesh.

This indicates that you selected all children of the translate module.

Now, copy the whole structure (edit > copy), select an empty matrix out connector of the hub module an activate edit > paste.

Paste the modules a second time to another matrix out connector of the hub.

Editing the Objects

Select one of the two pasted translate modules and switch to object mode.

Spatially divide the three cubes by moving the first selected subhierarchy.

Select the second translate module (in module mode or by activating edit > select > next sibling)

Translate this subhierarchy as well and select the cube (two times edit > select > children) afterwards.

Scale the cube by changing the size values in the inspector or by moving the scale handles (the little cubes attached to the move cones).

Scale another cube as well.

By activating view > projection > camera view you will see where the objects will appear in play mode. Note, that you still can change the object's positions but are no longer able to navigate (to prevent an accidentally change of the camera view).
