preferences
 
 

Activating window > show preferences opens the preferences window which lets you change certain behaviours of kodisein.

Screen Sizes

Your prefered screen sizes for window and fullscreen mode.

Modulation Keys

The modulator control keys.

Default Project

Defines, which project will be loaded when you start kodisein.

If 'last project is default' is activated, kodisein will open the project which was open when you closed kodisein. Otherwise it will load the file you specified in the 'default project file' field.

Saving Preferences

Activating the save button will store the preferences together with the menu shortcuts in the preferences file. It is located in the application's folder:

kodisein.app/Contents/MacOS/kodisein.cfg

To restore the default preferences and shortcuts, simply delete this file before starting kodisein.

Note, that the preferences will be saved automatically when you quit kodisein.
