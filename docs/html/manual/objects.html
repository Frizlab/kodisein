objects & attributes
 
 

Objects

Objects define what you will see in your scene. There are currently the following object modules available:

Geometric Primitives

  • Cube
  • Sphere
  • Cylinder
  • Disk
  • Landscape
  • Text
  • Face
  • Point

Other Objects

  • Camera & Light
  • L-System
  • Particle Emitter, Particle and Vertex

Attributes

An attribute defines the appearance/look of a certain object in play mode. There are currently the following attribute modules available:

  • Color
  • Texture
  • Texture (file)
  • Texture Matrix
  • Material
  • Fog
  • Blend
  • Size

Attribute Stack

Each attribute module has two connectors, an attribute out and an attribute in connector. You may apply more than one attribute to an object by building a stack of attribute modules.

Attribute Hub

An attribute hub module lets you apply a certain attribute(stack) to a collection of objects. All the objects that are connected to an attribute hub's matrix out connector share it's attributes.
